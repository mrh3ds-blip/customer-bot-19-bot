# تنیتاستس

این سرویس به صورت خودکار توسط ربات‌ساز ساخته شده است.

- نوع: رزرو / نوبت‌دهی بله
- امکانات: اتصال وبهوک بله، تقویم رزرو و نوبت‌دهی، گزارش‌گیری، چند ادمین، پرداخت کارت‌به‌کارت و تایید رسید
- پشتیبانی: تیدیددیدیدی

## اگر قالب تلگرام است
ادمین ربات با /start منوی مدیریت را می‌بیند و می‌تواند محصولات/خدمات، متن‌ها، سوال‌های فرم، کارت‌به‌کارت، لینک پرداخت و مرچنت زرین‌پال را تغییر دهد.

## اگر قالب اینستاگرام است
ساخته شدن سرویس به معنی آماده شدن سرور و Webhook است؛ داخل خود پیج اینستاگرام تا قبل از اتصال رسمی Meta تغییری دیده نمی‌شود.

راهنمای مشتری بعد از deploy:

- صفحه راهنما: /instagram/setup
- متن راهنما: /instagram/guide
- Webhook: /instagram/webhook

شرایط پیج اینستاگرام:

1. پیج باید Professional باشد: Business یا Creator.
2. پیج خصوصی قابل استفاده نیست؛ Professional Account عمومی است.
3. برای این نسخه، پیج باید به Facebook Page و Meta Developer App وصل شود.
4. از داخل ربات‌ساز اصلی روی «اتصال خودکار اینستاگرام» بزنید و در صفحه رسمی Meta/Facebook اجازه دسترسی را تأیید کنید.
5. سیستم اطلاعات لازم را خودکار دریافت و فعال‌سازی سرور را انجام می‌دهد.
6. بعد از اتصال، با یک اکانت دیگر به پیج دایرکت تست بفرستید.

## اگر قالب بله است

این سرویس یک بازوی بله واقعی با Webhook می‌سازد. بعد از بالا آمدن سرویس، Webhook بله خودکار روی مسیر /bale/webhook/... تنظیم می‌شود.

روش مدیریت برای مشتری:

1. داخل بله روی لینک بازو بزند یا username بازو را باز کند.
2. دستور /start را بفرستد.
3. اگر ادمین به صورت خودکار تنظیم شده باشد، دستور /claim کد-فعال‌سازی را که ربات‌ساز تحویل داده داخل بازو ارسال کند.
4. بعد از فعال شدن مدیریت، با /admin، /stats، /broadcast متن، /welcome متن، /item عنوان | قیمت | توضیح، تنظیم کارت‌به‌کارت و /reply کد پیام بازو را مدیریت کند.

## نکته پرداخت آنلاین
مرچنت/API زرین‌پال داخل پنل ذخیره می‌شود و پرداخت آنلاین با callback خودکار فعال است. Callback: /payment/zarinpal/callback
